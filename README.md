# Flashely Panel v3.0 — Hyfo Technology

Mobile-first Discord/Node.js bot management panel.
Inspired by Pterodactyl. Built for Termux + Desktop.

## Quick Start

```bash
npm install
node index.js
# Open http://localhost:3000
```

## Default Login
- Username: admin
- Password: admin123

## Pages
- /dashboard   — Server list & quick actions
- /console     — Terminal, charts, server controls
- /files       — File manager with code editor
- /logs        — Server logs with search
- /startup     — Startup commands, npm packages
- /database    — Database manager
- /environment — .env editor
- /stats       — Full statistics with live charts
- /domains     — Port info and reverse proxy config
- /backups     — Create and manage backups
- /settings-server — Server settings and danger zone
